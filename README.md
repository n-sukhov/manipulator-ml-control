# manipulator-ml-control

## Решение при помощи задачи регрессии

#### 2D-случай
Пусть манипулятор имеет $n$ сочленений. Обозначим множество углов поворота сочленений манипулятора: $\Theta=\{\theta_1, \theta_2,\dots, \theta_n\}$ и множество пар координат сочленений: $J=\{(x_1,y_1), (x_2,y_2), \dots,(x_n,y_n)\}\subset\R\times\R$. Функция для решения прямой кинематики для робота с заранее заданными длинами звеньев: $f_{fk}:\theta\rightarrow J$.

![Схема четырехзвенного робота-манипулятора](manipulator_4_links.svg)

Обозначим также множество точек, в которые должны прийти определёные сочленения робота $G$ (не все, а, например, только конечные). Индексы соответствуют номерам сочленений, к которым относятся целевые точки: $G=\{(x_i,y_i), (x_k,y_k), \dots,(x_s,y_s)\}\subset\R\times\R,\ |G|\leq n$. И расстояния между целевыми точками и сочленениями: $L=\{l_i, l_k,\dots,l_s\},\ |L|=|G|$. При этом расстояние между двуми точками из $G$ для соседних сочленений должно равняться длине звена, их соединяющего.
